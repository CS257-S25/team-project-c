# UFO Sightings CLI Tool

Users can filter ufo sightings by year and shape.

## Year

The dataset contains data from 1941 to 2013.

## Shape

The dataset categories ufos into the following list and other unkown catagories.

- changing
- cigar
- circle
- cylinder
- diamond
- disk
- egg
- fireball
- formation
- light
- oval
- rectangle
- sphere
- triangle
- teardrop
